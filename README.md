# CI/CD Pipeline using GitHub Actions

## Description
This project demonstrates a CI/CD pipeline that automatically deploys a web application using GitHub Actions.

## Tools Used
- Git
- GitHub Actions
- GitHub Pages
- HTML

## Workflow
1. Code pushed to main branch
2. GitHub Actions pipeline triggered
3. Website deployed automatically
4. Live site updated without manual steps
